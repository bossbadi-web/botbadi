Privacy Policy

By having botbadi in your Discord server or running its commands, you agree to the following privacy policies:

Data collected

Discord data collected from the application:

  • The server, channel, author, and message contents of every command run
    • This is used to provide analytics and errors to the developer.
    • This is stored temporarily on the disk (typically less than a day).
  • The channel, author, message contents of every deleted/edited message
    • This is used for the snipe and editsnipe commands.
    • This is stored temporarily in memory until the application restarts (typically once every few days).
  • The user ids of every user that has configured user settings in the application
    • This is stored permanently in the database.
  • The server ids of every server that has configured server settings in the application
    • This is stored permanently in the database.

Data visibility

Your recently deleted and edited messages can be seen by members in your server by running the corresponding snipe and editsnipe commands unless message sniping is disabled in the server settings.

The developer does not sell or release data to the public containing sensitive information including usernames and ids. Any data shared will always remain anonymous, unless permitted by the owners of said data. For example, if the developer discovers a humorous interaction between the user and the bot, it may be publicly shared in the application's support Discord server after removing sensitive information, if needed.

Data removal

  • Server settings are automatically deleted when the bot is removed from the server.
  • Contact the developer to have your user settings and other Discord data deleted.
